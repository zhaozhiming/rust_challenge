## 算法思路

#### 深度优先搜索

从第一个节点开始，采用深度优先搜索的方式进行遍历，记录访问节点的路径，直到倒数第二个节点时将本次访问路径记录到结果中。

## 复杂度分析

* 时间复杂度：O(2N∗N)
* 空间复杂度：O(2N∗N)
